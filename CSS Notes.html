Cascading Style Sheets is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML

Cascading Style Sheets

Cheat Sheet and Notes

Selectors

element Select elements of specific type
#id Select an element with a specific id
.class Select an element with a specific class

Text properties

color: blue; Set the text color
#font-size: 10px; Set the font size
font-family: Arial; Set the font family
background-color: red; Set the background color

Box Properties

margin: 10px; Set the outer spacing around an element
padding: 15px; Set the space between content and border
border: 1px solid black; Set the border style and color
width: 200px; Set the width of an element
height: 100px; Set the height of an element

Display Properties

display: block; Element takes full width, starts on a new line
display: inline; Element flows within text, no new line
display: inline-block; Like inline, but allows width/height
display: flex; Turns container into a flexbox
display: none; Hide the element

Positioning Properties

position: static; Default position (no special positioning)
position: relative; Position relative to its normal position
position: absolute; Position relative to the nearest positioned ancestor
position: fixed; Position relative to the viewport
z-index: 10; Set the stack order of elements

Units

px Pixels — fixed size units
% Percentage relative to parent
em Relative to the font-size of the element
rem Relative to the font-size of the root element
vh / vw Viewport height/width units
