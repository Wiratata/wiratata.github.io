JavaScript, often abbreviated as JS, is a programming language and core technology of the World Wide Web, alongside HTML and CSS.

JavaScript

Cheat Sheet and Notes

Variables & Data Types

let x = 5; Declare a block-scoped variable
const y = "hello"; Declare a constant variable
typeof x; Check the data type of a variable
Number, String, Boolean Common data types in JavaScript
null, undefined Special values representing "empty" or "not assigned"

Operators

+ Addition or string concatenation
=== Strict equality comparison
!= Not equal
++ / -- Increment / Decrement a value
+= Add and assign

Control Structures

if (x > 5) { ... } Execute code if the condition is true
else { ... } Run this block if the if-condition is false
for (let i = 0; i < 5; i++) Loop from 0 to 4
while (x < 10) Repeat while the condition is true
switch (value) { case ... } Multiple condition checks

Functions

function greet() { ... } Declare a function
greet(); Call or invoke the function
return value; Return a value from a function
const add = (a, b) => a + b; Arrow function syntax
parameters & arguments Inputs passed into functions

Array Methods

arr.push(4); Add item to the end of the array
arr.pop(); Remove the last item
arr.length; Get the number of items
arr.map(fn); Create a new array by transforming each item
arr.includes(2); Check if an item exists in the array
